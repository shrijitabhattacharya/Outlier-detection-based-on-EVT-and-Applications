Code Files Description
======

* RHG_Figures_Code_Rev.R to replicate all results in Section 3 of the paper.
* RHG_Data_Code_Rev.R to replicate all results in Section 4 of the paper.
* multivariate_outlier_generation.R to replicate all results in Section of the paper.
* RHG_Source_Code_Rev.R contains all source files needed to run RHG_Figures_Code_Rev.R, RHG_Data_Code_Rev.R and multivariate_outlier_generation.R
